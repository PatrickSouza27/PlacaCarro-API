```https
  https://localhost:{0000}/swagger/index.html
```

#### Valor
```https
  https://localhost:{0000}/{NumeroPlaca}
```

| string placa   | Tipo       | Descrição                           |
| :---------- | :--------- | :---------------------------------- |
| IZO1880 | `string` | Coloque uma placa valida |
